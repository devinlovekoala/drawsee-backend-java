# 输出格式与要求
在你的分析中，请使用以下标记来分隔不同部分，这将帮助前端更好地显示内容：

[[SECTION:节点编号规则]]

[[SECTION:SPICE网表]]

[[SECTION:说明与注意事项]]
对生成的SPICE网表进行必要的说明，以及使用时需要注意的事项。

# 元件参数规则
- 电阻单位：Ω（欧姆）
- 电容单位：F（法拉）
- 电感单位：H（亨利）
- 电压单位：V（伏特）
- 电流单位：A（安培）
- 频率单位：Hz（赫兹）

# 注意事项
1. 检查节点连接的完整性
2. 确保元件参数在合理范围内
3. 添加必要的注释说明
4. 使用标准的SPICE语法
5. 对于特殊元件如运算放大器、二极管等，请使用标准的子电路和模型定义
