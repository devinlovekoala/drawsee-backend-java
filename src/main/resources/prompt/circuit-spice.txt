# 转换规则
1. 节点编号规则
   - 接地节点使用0
   - 其他节点从1开始递增编号
   - 相连的端口共用同一个节点编号

2. 元件命名规则
   - 电阻：R + 编号
   - 电容：C + 编号
   - 电感：L + 编号
   - 电压源：V + 编号
   - 电流源：I + 编号
   - 二极管：D + 编号
   - 晶体管：Q + 编号
   - 运算放大器：X + 编号

3. 元件参数规则
   - 电阻单位：Ω（欧姆）
   - 电容单位：F（法拉）
   - 电感单位：H（亨利）
   - 电压单位：V（伏特）
   - 电流单位：A（安培）
   - 频率单位：Hz（赫兹）

4. 特殊元件处理
   - 运算放大器使用子电路定义
   - 二极管和晶体管使用模型定义
   - 受控源使用标准SPICE语法

# 输出格式
1. 网表标题和注释
2. 元件定义（每行一个）
3. 控制语句
4. 结束语句(.end)

# 注意事项
1. 检查节点连接的完整性
2. 确保元件参数在合理范围内
3. 添加必要的注释说明
4. 使用标准的SPICE语法
