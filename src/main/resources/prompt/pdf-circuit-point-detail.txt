# 典型场景示例

## 示例1：元器件类要点（如"555定时器工作原理"）
```markdown
## 555定时器工作原理

555定时器是本实验的核心器件，它能产生精确的时序信号。理解它的工作模式对于正确搭建电路和调试至关重要。

### 关键知识点

**1. 三种工作模式**
- 单稳态模式：产生单次脉冲，脉宽由 $t = 1.1RC$ 决定
- 双稳态模式：作为RS触发器使用，输出保持稳态
- 无稳态模式（本实验使用）：自激振荡产生方波，频率 $f = \frac{1.44}{(R_1+2R_2)C}$

**2. 引脚功能**
- 2脚（触发）：低于1/3 VCC时触发输出高电平
- 6脚（阈值）：高于2/3 VCC时使输出变低电平
- 7脚（放电）：内部晶体管，控制电容充放电

### 实验操作要领

1. 按电路图连接R1、R2和C到555的6、7脚，形成RC充放电回路
2. 用万用表测量2脚和6脚电压，验证触发和阈值门限是否为VCC的1/3和2/3
3. 用示波器在3脚观察输出方波，测量频率并与计算值比较
4. 调节可变电阻观察频率变化，验证公式关系

### 重要提示

❗ **电源电压范围**
555定时器工作电压5-15V，实验中使用+12V供电，不要接错电源极性否则芯片会损坏

💡 **常见问题**
如果无输出：检查4脚（复位）是否接到+VCC，该脚低电平会禁止输出

✅ **实用技巧**
先用较大电容（如10μF）测试，波形周期长便于观察；验证正确后再换小电容提高频率
```

## 示例2：测量类要点（如"示波器触发设置"）
```markdown
## 示波器触发设置

触发功能是示波器最关键的功能，设置不当会导致波形抖动无法观测。掌握正确的触发方法能让你快速捕获稳定的波形。

### 关键知识点

**1. 触发原理**
示波器通过检测信号的特定电平和斜率来同步显示，使周期信号在屏幕上静止。触发电平决定了波形从哪个点开始绘制。

**2. 触发模式选择**
- 自动触发（Auto）：无触发信号时也显示波形，适合初步观察
- 常规触发（Normal）：必须满足触发条件才显示，适合精确测量
- 单次触发（Single）：捕获一次触发事件，适合瞬态信号

### 实验操作要领

1. 将触发源设置为你要观察的通道（CH1或CH2）
2. 触发方式选择"边沿触发"，斜率选"上升沿"（↑）
3. 旋转"LEVEL"旋钮，将触发电平线调到波形幅度的中间位置
4. 观察波形稳定后，微调触发电平使波形起始点对齐屏幕左侧刻度

### 重要提示

❗ **触发电平位置**
触发电平应设在信号幅度的20%-80%范围内，太靠近噪声电平会造成虚假触发

💡 **常见问题**
波形一直跑动不稳定：检查触发源是否选对，触发电平是否在信号范围内

✅ **实用技巧**
对于复杂波形，可以按"RUN/STOP"暂停显示，然后用"MEASURE"自动测量功能读取参数
```
