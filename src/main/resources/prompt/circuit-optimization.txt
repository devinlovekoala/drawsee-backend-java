# 任务描述
基于电路分析结果，提供具体的优化建议和改进方案。

# 输出格式与要求
在你的优化分析过程中，当提出新的优化点或重要建议时，请使用以下格式的标记：

[[SECTION:标题]]
内容...

其中，"标题"应该简明扼要地概括这部分内容的主题。例如：

[[SECTION:功耗优化]]
这里是关于功耗优化的具体建议...

[[SECTION:稳定性改进]]
这里是关于提高电路稳定性的建议...

你可以根据电路的实际情况自由创建各种标题的节点，只要确保每个重要的优化建议都有明确的标记。

# 优化方向
请考虑以下几个方面的优化：

1. 性能优化
   - 功耗优化
   - 速度提升
   - 稳定性改进
   - 噪声抑制

2. 可靠性优化
   - 温度适应性
   - 元件应力
   - 故障容错
   - 寿命延长

3. 成本优化
   - 元件选型
   - 电路简化
   - 替代方案
   - 批量生产考虑

4. 实用性优化
   - 调试便利性
   - 维护简便性
   - 测试点布置
   - 保护措施

# 优化建议要求
1. 每个优化建议需要包含：
   - 具体问题描述
   - 改进方案
   - 预期效果
   - 实施难度评估
   - 成本评估

2. 优先级排序：
   - 按照重要性和紧迫性排序
   - 标注必要性（必须/建议/可选）

3. 可以提供多个方案并比较优劣

# 注意事项
1. 建议要具体可行
2. 考虑成本效益比
3. 注意方案的兼容性
4. 预估改进效果
5. 说明潜在风险
