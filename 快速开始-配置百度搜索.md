# 快速开始：配置百度AI搜索

## 1分钟快速配置

### 步骤1: 获取API Key

1. 访问 https://qianfan.cloud.baidu.com/
2. 注册/登录账号
3. 控制台 → API Key → 创建
4. 服务选择：**千帆AppBuilder**
5. 复制API Key（格式：`bce-v3/ALTAK-***`）

### 步骤2: 配置项目

**方法A：环境变量（推荐）**
```bash
export SEARCH_ENABLED=true
export SEARCH_API_KEY="bce-v3/ALTAK-你的密钥"
```

**方法B：配置文件**
编辑 `application.yaml`：
```yaml
drawsee:
  search:
    enabled: true
    api-key: bce-v3/ALTAK-你的密钥
```

### 步骤3: 启动项目

```bash
mvn spring-boot:run
```

查看日志确认搜索功能已启用：
```
INFO - Web搜索功能已启用，使用百度AI搜索API
```

## 常见问题

**Q: API Key格式是什么？**
A: `bce-v3/ALTAK-***` 开头，约40-50个字符

**Q: 如何获取免费额度？**
A: 新用户注册百度智能云自动获得免费额度

**Q: 搜索不生效？**
A: 检查：
1. API Key是否正确复制（注意不要有空格）
2. `enabled: true` 是否设置
3. 网络是否能访问百度云

**Q: 如何暂时关闭搜索？**
A: 设置 `SEARCH_ENABLED=false` 或 `enabled: false`
